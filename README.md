## 🛠️ Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/multi-platform-listing.git
```

2. Install dependencies
```bash
pnpm install
```

3. Set up environment variables
```bash
cp .env.example .env.local
```

4. Start the development server
```bash
pnpm dev
```
